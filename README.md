# MCP Ecosystem

Репозиторий содержит MCP серверы и автономные AI агенты, разработанные в соответствии с [MCP Ecosystem Guidelines](MCP_ECOSYSTEM_GUIDELINES.md).

## HALv1 - Автономный AI агент для Telegram

**HALv1** — интеллектуальный агент, интегрированный с экосистемой MCP для выполнения сложных задач через Telegram.

### Основные возможности

- **Обработка сообщений Telegram** — индексация, поиск, анализ чатов
- **Планирование и выполнение задач** — многошаговые задачи с LLM-планированием
- **Безопасное выполнение кода** — изоляция в Docker контейнерах
- **Система памяти L0-L∞** — эпизодическая и долгосрочная память с графовой структурой
- **Семантический поиск** — векторный поиск по сообщениям и контексту
- **Интеграция с MCP серверами** — унифицированное взаимодействие через MCP протокол

### Используемые MCP серверы

- **Memory MCP** — графовая память, семантический поиск, управление контекстом
- **TradingView MCP** — анализ криптовалютного рынка, технические индикаторы
- **Binance MCP** — исполнение торговых операций, управление портфелем
- **Backtesting MCP** — тестирование торговых стратегий на исторических данных
- **Shell MCP** — выполнение кода в изолированной среде
- **Supervisor MCP** — мониторинг и управление MCP серверами, метрики
- **Learning MCP** (в разработке) — адаптация и обучение на опыте
- **Policy MCP** (в разработке) — политики безопасности и контроля рисков
- **Orchestrator MCP** (в разработке) — координация задач между MCP серверами
