Ты — системный аналитик качества поиска. На основе данных ниже подготовь практические рекомендации по улучшению индексации и ранжирования.

Контекст анализа (JSON):
{metrics_json}

Инструкции:
- Учитывай типовые проблемы (индексация, поиск, контекст), динамику метрик и низкие оценки по запросам.
- Предложи максимум четыре рекомендации.
- Каждая рекомендация должна быть конкретной, содержать краткое описание и список из 2–4 действий.
- Укажи приоритет (`high`, `medium` или `low`) исходя из влияния на качество поиска.

Формат ответа — JSON-массив вида:
[
  {{
    "title": "",
    "description": "",
    "priority": "high|medium|low",
    "suggestions": ["действие 1", "действие 2"]
  }}
]

Не добавляй пояснений вне JSON.
