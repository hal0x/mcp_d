# Исправление проблемы с демо-торговлей Binance

## Проблема
Ошибка `APIError(code=-2015): Invalid API-key, IP, or permissions for action` при попытке использовать демо-торговлю.

## Решение

### 1. Установите переменную окружения
```bash
export BINANCE_DEMO_TRADING=true
```

### 2. Используйте правильные URL для демо-торговли
- **Futures Demo**: `https://demo-fapi.binance.com`
- **Delivery Demo**: `https://demo-dapi.binance.com`

### 3. Проверьте API ключи
Убедитесь, что используете API ключи, созданные специально для демо-торговли на Binance Futures.

### 4. Проверьте разрешения
API ключ должен иметь разрешения на:
- Futures Trading
- Чтение информации об аккаунте
