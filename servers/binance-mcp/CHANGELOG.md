# Changelog

Все значимые изменения в проекте Binance Native MCP Server будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
и проект следует [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Добавлено
- Полная поддержка Binance API для торговли и анализа рынка
- MCP сервер с 30+ инструментами для работы с Binance
- Поддержка демо и продакшн режимов
- Интеграция с Redis для кэширования
- Поддержка PostgreSQL для хранения данных
- Telegram уведомления для алертов
- Система управления рисками
- Батчевые операции для массовой торговли
- Технические индикаторы (RSI, ADX)
- Автоматическое управление стоп-лоссами
- Интеграция с Supervisor MCP (metrics/facts), экспоненциальный backoff для Binance API, конфигурируемые retries

### Изменено
- Улучшена архитектура проекта с четким разделением ответственности
- Оптимизирована обработка ошибок и логирование
- Обновлена документация и примеры использования
- Обновлена зависимость `mcp` до `>=1.18,<2.0` и добавлена выдача `structuredContent` для обратной совместимости с новыми клиентами

### Исправлено
- Исправлен импорт Field в models.py
- Исправлены типы возврата в config.py
- Исправлено дублирование имен переменных в services.py
- Добавлены недостающие аннотации типов
- Улучшено форматирование кода с помощью Black

## [1.0.0] - 2024-12-19

### Добавлено
- Первоначальный релиз Binance Native MCP Server
- Базовая функциональность для работы с Binance API
- Поддержка спотовой и фьючерсной торговли
- Маржинальные операции и OCO ордера
- Система управления портфелем
- Инструменты для анализа рынка
